# 试用本服务三步走

## 第一步

设定你的主页地址到你的目标域名: 比如本人的 [blog.crhan.com][]

## 第二步

添加 [http://linode.crhan.com:3000/ruhoh][] 到 [WebHooks][]

## 第三步

添加目标域名的 A 记录到 `173.255.218.208`, AAAA 记录到 `2600:3c01::f03c:91ff:feae:6ad0`

## 第四步

推送一个更新到仓库或者直接点测试
